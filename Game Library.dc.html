Adding a New Game

No backend or admin panel. Add files to the repo and push — Vercel redeploys automatically.

Steps
1Add the game folder to public/games/[slug]/
2Make sure the game has index.html in that folder
3Add a thumbnail to public/thumbnails/[slug].png
4Add the game entry to src/data/games.ts
5Commit and push to GitHub
6Vercel will redeploy automatically
Folder structure
public/
├── games/
│   └── my-game/
│       ├── index.html
│       ├── game.js
│       ├── style.css
│       └── assets/
└── thumbnails/
    └── my-game.png
Example entry in games.ts
{
  title: "My Game",
  slug: "my-game",
  description: "A short description of the game.",
  thumbnail: "/thumbnails/my-game.png",
  url: "/games/my-game/index.html",
},
